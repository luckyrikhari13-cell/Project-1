## 📋 About the Project

ShareOn Enterprise is an **AI Personal Branding Agent** built for solopreneurs, founders, coaches, consultants, agencies, and educators. This landing page was built as **Project 1** to showcase the product's core value proposition and drive user signups.

### Target Users
- 🚀 Founders building thought leadership
- 🎓 Coaches repurposing frameworks into content
- 💼 Consultants turning client work into authority
- 🏢 Agencies scaling content for multiple clients
- 📚 Educators transforming course content into posts

---

## ✨ Features of the Landing Page

| Section | Description |
|---|---|
| **Navbar** | Sticky navigation with blur backdrop and CTA button |
| **Hero** | Animated gradient glow orb, headline, and dual CTAs |
| **Social Proof** | Trust bar with creator/company logos |
| **How It Works** | 3-step process — Upload → Generate → Publish |
| **Features Grid** | 6 glassmorphism feature cards |
| **Who It's For** | 5 persona cards |
| **Pricing** | Spark, Creator, and Pro plans with feature lists |
| **CTA Banner** | Full-width conversion section with gradient background |
| **Footer** | Logo, links, socials, and copyright |

---

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, Grid, Flexbox, animations
- **Vanilla JavaScript** — Intersection Observer for scroll animations
- **Google Fonts** — Space Grotesk (headings) + Inter (body)

> No frameworks. No build tools. Single `index.html` file.
